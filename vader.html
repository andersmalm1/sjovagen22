USB-dataloggning: Väder-stationen kan ansluta till dator via USB för kontinuerlig datainsamling. Mjukvaran på CD möjliggör real-time övervakning och historisk data-analys. Detta är en professionell väderstation med full dokumentation för integration i hemautomationssystem.
⚠ Känd defekt — Anemometer:

Vindhastighetsgivaren (anemometer, kopp-typ) är trasig och har kärvat fast. Den mäter inte vindstyrka korrekt. Övrig väderdata (temperatur, fukt, lufttryck, nederbörd, vindriktning) loggas normalt.

Reparationsmöjligheter:
· Försöka försiktigt rengöra och smörja anemometerns lager
· Byte av anemometer-enhet (kan köpas separat från tillverkaren)
· Alternativ: Ignorera vindstyrka-datan, logga övrig väderdata normalt

Integrationsvägar (prioriterad ordning)
Väg 1: USB-kabel direkt från väder-stationRekommenderat. Anslut USB-kabel från väder-stationens display-enhet direkt till dator eller Raspberry Pi. Mjukvaran från CD installeras på datorn för att läsa all mätdata (temp, fukt, vind, nederbörd, lufttryck) i realtid. Data kan loggas till CSV/databas.
Mjukvaran från CD✓ Tillgänglig · innehåller drivrutiner för USB + loggningsprogram · Kontrollera Windows/Mac/Linux-kompatibilitet · utforska innehållet
RTL-SDR + rtl_433Alternativ: USB-dongel som lyssnar på 433 MHz · avkodar Viking-protokoll från sensorerna · lämplig för Raspberry Pi · ca 100–200 kr om USB-vägen inte fungerar
Raspberry PiKör USB-drivrutin (från CD) eller rtl_433 · loggar temperatur, fukt, vind till lokal databas/MQTT
StatusVäg 1 (USB): kan startas omedelbar när CD:n utforskas. Väg 2 (RTL-SDR): planerad.
Nästa steg: Undersök mjukvaru-CD:n för att se vilka operativsystem som stöds, vilka USB-drivrutiner som krävs och vilka möjligheter till dataloggning som finns. Mjukvaran kan möjliggöra omedelbar kontinuerlig väderdata-insamling utan att köpa RTL-SDR-hårdvara.
