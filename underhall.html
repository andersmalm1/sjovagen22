01 / Rutiner

Regelbundet underhåll

Värmebaronen Pellmax UB med Viking Bio-brännare · frekvenser baserade på tillverkarens anvisningar

Var 1–2 vecka
🔥 Brännare — löpande rengöring
1.Stäng av brännaren och vänta tills den svalnat (~30 min)
2.Töm brännarkoppens askskåpa med askrakan
3.Rengör det inre brännarröret från aska och slagg
4.Kontrollera att rosten sitter rätt — felplacerat rost förlänger tändtiden
5.Kontrollera att pelletsmatningen flödar fritt i fallröret
6.Starta om och verifiera normal drift
Varannan vecka
🛢 Panna — askbehållare
1.Töm pannans askbehållare
2.Förvara askan i plåthink med tätt lock på obrännbart underlag
3.Låt askan svalna minst 24 h innan den töms i sopkärl
1 gång per säsong
🔧 Panna — säsongsservice
1.Sota pannans värmeytor och rökgångar med sotviska
2.Rengör rökgasväxlaren (turbulatorerna)
3.Kontrollera tätningar och luckor
4.Kontrollera tändelementet — byt vid behov
5.Kontrollera flamvakten — rengör vid behov
6.Justera förbränningsvärden (luft/bränsle) om rökgastempen ökat
Vid varje påfyllning
📋 Pelletslager — loggning
1.Fotografera/spara leveranssedeln
2.Logga leveransen: datum, ton, kr/ton, leverantör
3.Exportera och uppdatera pellets_data.json i GitHub
Tips — när ska man sota? Rökgastemperaturen är det bästa måttet. Nota temperaturen efter en nysotad panna. När den stigit 40–50°C över detta värde är det dags att sota igen — oavsett intervall. En digital rökgastermometer med larm förenklar detta.
⚠ Viktigt: Bryt alltid spänningen och lossa kontakten på brännaren innan rengöring, service eller demontering. Låt brännaren svalna ordentligt — risk för brännskada.
02 / Pelletsmatning

Matarskruv & pelletsflöde

Motordriven skruv i lagrets botten · slang från lager till brännare · vanliga problem och förebyggande åtgärder

🔄
Fyll skruven inför säsongsstart
Efter längre uppehåll (sommar) är matarskruven tom. Starta brännaren i manuellt testläge och låt skruven fylla sig innan normal drift startar — annars kan brännaren larmfela vid "misslyckad start".
🪵
Kontrollera fallröret regelbundet
Fallröret (från skruven ner till brännaren) kan proppa igen med pelletsrester eller damm. Rensa det vid varje brännärrengöring. En propp syns som ojämn förbränning eller misslyckad start.
📐
Kontrollera att innerröret sitter rätt
Om det inre brännarröret hänger snett bildas en "hylla" i övergången mellan fallröret och brännarröret. Pellets fastnar och skapar ojämn matning. Kontrollera rörets läge vid varje rengöring.
💨
Säkerställ god tilluft till pannrummet
Brännaren och fläkten drar ganska mycket luft. Stängda ventiler i pannrummet ger dålig förbränning och snabbare sotbildning. Kontrollera att ventilationsgallret är öppet och fritt från damm.
⚡
Kalibrera matarskruvens kapacitet
Brännaren har en inbyggd kalibreringsfunktion för bränslemängd. Välj effektläge, starta test, samla upp och väg pelletarna som matas ut under 1 minut. Mata in vikten i brännaren för korrekt effektstyrning.
🏷
Pelletskvalitet påverkar rengöringsintervallet
Pellets av klass 1 (svensk standard) ger minst aska och slagg. Sämre kvalitet eller fel diameter (6 vs 8 mm — kontrollera inställningen i brännaren) ger snabbare sotbildning och tätare rengöring.

⚡ Kopplingschema — Raspberry Pi GPIO

HC-SR04 ECHO-pin ger 5V signal. Raspberry Pi:s GPIO tål max 3,3V. Använd spänningsdelare med två motstånd:

Komponenter: 1 × 1 kΩ resistor · 1 × 2 kΩ resistor · kopplingskabel

HC-SR04 VCC → Pi pin 2 (5V)
HC-SR04 GND → Pi pin 6 (GND)
HC-SR04 TRIG → Pi GPIO23 (pin 16, 3,3V OK direkt)
HC-SR04 ECHO → 1kΩ → Pi GPIO24 (pin 18)
                   ↓
                 2kΩ → Pi GND
(spänningsdelare 5V→3,3V: R1=1kΩ, R2=2kΩ)
Installation — engångssteg
🔧 Montering i pelletslagret
1.Montera givaren centrerat i taket på pelletslagret, riktad rakt nedåt
2.Säkerställ att givaren sitter minst 10 cm ovanför pelletsytan när lagret är fullt
3.Kabelvägen från lagret till Pi: dra längs vägg, skydda mot pelletsdam med tätningslista
4.Bygg spänningsdelare (1kΩ + 2kΩ) på lödplatta eller breadboard nära Pi:n
5.Anslut enligt kopplingsschemat ovan
6.Kör kalibreringsscipt och verifiera avläsning mot känd referens
Kontroll — vid behov
🧹 Underhåll
1.Blås rent givarytan från pelletsdam med tryckluft — damm ger felaktiga avläsningar
2.Kontrollera att inga pellets fastnat direkt under givaren
3.Verifiera att kabelanslutningar sitter fast (vibrationer från skruv kan lossa kontakter)
4.Jämför avläst nivå mot visuell inspektion vid påfyllning
🐍 Python-script — pellets_niva.py

Installera bibliotek: pip3 install RPi.GPIO --break-system-packages

import RPi.GPIO as GPIO import time, json, statistics # GPIO-pinnar (BCM-numrering) TRIG = 23 ECHO = 24 # Lagergeometri Sjövägen 22 LAGER_HOJD_CM = 200.0 # lagrets totala höjd (cm) LAGER_BREDD = 1.4 # m LAGER_LANGD = 2.0 # m PELLETS_KG_M3 = 650 # kg/m³ bulk GIVARE_OFFSET = 12.0 # cm från lagrets tak till givaren def mat_avstand_cm(n_sampel=11): """Returnerar medianen av n avståndsmätningar i cm""" GPIO.setmode(GPIO.BCM) GPIO.setup(TRIG, GPIO.OUT, initial=GPIO.LOW) GPIO.setup(ECHO, GPIO.IN) matningar = [] for _ in range(n_sampel): GPIO.output(TRIG, True) time.sleep(0.00001) GPIO.output(TRIG, False) start = time.time() stopp = time.time() while GPIO.input(ECHO) == 0: start = time.time() while GPIO.input(ECHO) == 1: stopp = time.time() matningar.append((stopp - start) * 34300 / 2) time.sleep(0.1) GPIO.cleanup() return round(statistics.median(matningar), 1) def berakna_niva(avstand_cm): """Beräknar lagernivå i ton från avståndsmätning""" djup_cm = LAGER_HOJD_CM - (avstand_cm - GIVARE_OFFSET) djup_m = max(0, min(djup_cm / 100, LAGER_HOJD_CM / 100)) volym_m3 = djup_m * LAGER_BREDD * LAGER_LANGD ton = round(volym_m3 * PELLETS_KG_M3 / 1000, 2) pct = round(ton / 3.6 * 100, 1) return {"avstand_cm": avstand_cm, "djup_cm": round(djup_cm,1), "volym_m3": round(volym_m3,2), "ton": ton, "procent": pct} if __name__ == "__main__": avstand = mat_avstand_cm() niva = berakna_niva(avstand) print(json.dumps(niva, indent=2))
📐 Kalibrering — viktigt steg

Mät med måttband det faktiska avståndet från givaren till pelletsytan (eller till lagergolvet om tomt). Jämför med avläst avstand_cm. Justera GIVARE_OFFSET tills värdena stämmer.

Kontroll vid påfyllning: Notera ton enligt leveranssedel. Jämför med beräknat ton från givaren direkt efter påfyllning. Avvikelse >0,2 ton → justera LAGER_HOJD_CM eller PELLETS_KG_M3.
⚠ Vanliga problem och lösningar

Avläsning hoppar ±10–20 cm: Pelletsytan är ojämn. Använd median av 11 mätningar (redan inbyggt i scriptet). Kan också bero på pelletsdam på givarytan — rengör.

Alltid max-värde (215 cm): ECHO-pin får ingen signal. Kontrollera spänningsdelare och kablar. Mät spänning på ECHO-pin med voltmeter — ska vara 3,3V vid signal.

Script fryser: Pi väntar på ECHO-signal som aldrig kommer. Lägg till timeout i while-loopen (se avancerad version på GitHub: merbanan/hc-sr04).

Felaktigt ton-värde: Kontrollera GIVARE_OFFSET och LAGER_HOJD_CM mot faktiska mått med måttband.
